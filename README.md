# CRM Artisan

Next.js CRM application built with App Router, TypeScript, Tailwind CSS, and ShadCN.

## Stack

- **Next.js 16** — App Router, React Server Components
- **TypeScript**
- **Tailwind CSS v4**
- **ShadCN** — component library (`src/components/ui/`)

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Adding ShadCN Components

```bash
npx shadcn@latest add <component>
```
